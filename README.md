# Odin Recipes
## Overview
This project demonstrates my understanding of the core html elements and website file structures. The main page introduces the topic "Recipes" and provides links to three recipe web pages.

The recipe pages each contain an image and lists explaining the details of the given recipes.

Key skills used were implementing git commands and observing the git commit message conventions. The project repository is linked to my github website and the project is available to view [HERE](https://cormacsugrue.github.io/odin-recipes/)
